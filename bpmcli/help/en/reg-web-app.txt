COMMAND TYPE
    CI/CD commands

NAME
    reg-web-app - create/update an environment

DESCRIPTION
    Register new application settings or update existing

SYNOPSIS
    bpmcli reg-web-app <ENVIRONMENT_NAME> -u http://mysite.bpmonline.com -l administrator -p password

OPTIONS
    Name (pos. 0)	Environment(application) name
    
    --ActiveEnvironment     -a          Set a web application by default
    
    --Safe                  -s          Safe action in this environment

    --uri                   -u          Application uri

    --Password              -p          User password

    --Login                 -l          User login (administrator permission required)

    --Maintainer            -m          Maintainer name

EXAMPLE
    bpmcli reg-web-app <ENVIRONMENT_NAME> -u http://mysite.bpmonline.com -l administrator -p password
        creates new environment, named <ENVIRONMENT_NAME> or updates existing environment settings
