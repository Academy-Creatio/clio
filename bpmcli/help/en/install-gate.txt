COMMAND TYPE
    Service commands

NAME
    install-gate - installs a service package that expands bpmcli commands

DESCRIPTION
    Installs on the selected web application(website) a service package 
    "bpmcligate" that makes available all bpmcli's commands.

EXAMPLE
    bpmcli install-gate

    bpmcli install-gate -e <ENVIRONMENT NAME>
